**NeuralJack** turns your ChatGPT export into everything you need to move to Claude. Drop your OpenAI data ZIP into the app, pick your options, and get a **Memory Core**, organized conversation exports, and step-by-step import promptsвҖ”so you donвҖҷt leave your history or context behind.

---

## What it does

- **Parses ChatGPT data** вҖ” Cleans OpenAI data export for migration to Claude.
- **Builds a Memory Core** вҖ” Context document to paste into Claude.
- **Exports by project** вҖ” Markdown per conversation, grouped by ChatGPT project, ready for Claude.
- **Creates prompted guides** вҖ” Step-by-step migration prompts for Cowork or manual import.

---

## Quick start

1. **Export your ChatGPT data**  
   - ChatGPT вҶ’ Settings вҶ’ [Data Controls](https://chatgpt.com/#settings/DataControls) вҶ’ **Export data** вҶ’ download the ZIP.

2. **Open NeuralJack**  
    - Import the data ZIP or extracted folder.

3. **Customize Your Migration Path**
   - **Parse Your Data** вҖ” Pick an output folder.
   - **Select Projects** вҖ” Choose which projects to export; the Memory Core (or prompt) uses only these.
   - **Memory Core** вҖ” Optionally use Claude API to generate a Memory Core or export a prompt to run in Claude Cowork.
   - **Export** вҖ” Projects, conversations, and prompts are written to your output folder.

4. **Use the output**
   - Conversations sorted by project to import to Claude.
   - Prompt files to guide you and Claude to migrate data.
